1. Как открыть командную строку
Способ 1 - Зажать кнопки Win + R (Кнопка с иконкой виндоуса + R), ввести слово cmd и нажать Enter
Способ 2 - Зайти в меню Пуск => Найти папку Git => Открыть файл Git CMD

2. Команды
cd имя папки - переход по папке, которая есть по текущему пути
cls - очистить командную строку
Помните, что если вам нужно посмотреть информацию о проекте (репозитории) нужно находиться в папке с репозиторием

3. Работа с проектом (репозиторием)
	1. Чтобы скачать проект, нужно ввести следующее
		git clone https://github.com/AndrewKhukhrianskyi/Game-Novel-Project-Potato
	2. Чтобы посмотреть состояние репозитория нужно ввести команду git status. Красным будут отмечены не зафиксированные файлы
	3. Чтобы создать бранч (ветку) нужно прописать команду git checkout -b название ветки (слитно на английском или отделяете дефисами)
	4. Чтобы проверить, на какой ветке мы находимся, нужно прописать git branch. Выбранная ветка будет отмечена звездочкой
	и ярким цветом (это будет означать, что вы находитесь на текущей ветке)
	5. Переключаться между ветками можно через git checkout название существующей ветки
	6. Для добавления файлов в проект (репозиторий) пишем  git add . (Точка в конце = добавление всех новых файлов в бранч)
	7. Для зафиксирования файлов в бранче пишем следующее: git commit -m "Сообщение". Сообщение пишете в двойных кавычках
	8. Для отправки файлов в Github пишем: git push origin название ветки

4. Как мне добавить файлы?

Сделали домашку => Закинули файл в папку с проектом и подписали ее => git checkout -b имя ветки => git add . => git commit -m "Сообщение" => git push origin название ветки
